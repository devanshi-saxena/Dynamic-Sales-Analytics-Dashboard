# 📊 Dynamic Sales Analytics Dashboard

## 🚀 Overview
An interactive Excel dashboard built to monitor and analyze sales performance across multiple states. The dashboard dynamically updates charts and KPIs based on selected filters, enabling quick and efficient business analysis.

This project focuses on transforming raw sales data into actionable insights using Excel’s analytical and visualization capabilities.

---

# 🛠 Tech Stack
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

---

# 📌 Dashboard Highlights
✅ State-wise Sales Analysis  
✅ Top & Bottom Sales Executives  
✅ Target Achievement Tracking  
✅ Sales Trend Visualization  
✅ Interactive Filtering & Reporting  

---

# 📈 Business Insights Generated
- Identified top-performing sales executives
- Analyzed executives with low target achievement
- Compared sales performance across states
- Tracked overall sales trends and KPIs
- Evaluated performance gaps for improvement

# 🎯 Key Learning Outcomes
- Interactive dashboard creation
- Data storytelling through visualization
- KPI reporting and analysis
- Business-focused analytics
- Advanced Excel reporting techniques

# 💡 Project Goal
To build a dynamic and user-friendly sales analytics dashboard that helps businesses monitor performance and make data-driven decisions efficiently.
